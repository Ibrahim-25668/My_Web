About Dalilak Site

“Dalilak” is a website aimed at collecting and organizing all events held in Syrian cities — from festivals, exhibitions, cultural, sports, and artistic activities — in one place for easy access.

Site Objectives :

  • Create an accurate and updated database of events.
  • Facilitate the search process using smart filters.
  • Support communication between event organizers and visitors.
  • Contribute to promoting domestic tourism and local culture.

Our Team

The Dalilak website was developed as part of a university project to display events interactively using HTML, Bootstrap, and JavaScript to achieve an easy and engaging user experience.
